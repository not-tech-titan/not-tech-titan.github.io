Zed Ops: Java Edition

Now with pre-rendered sprites!

Zed Ops: Java Edition is a infinite zombie shooter survival inspired by Black Ops Zombies on the DS. Players face countless hordes of undead, navigating through the map, unlocking new areas, and gathering new weaponry to survive as long as possible. The game emphasizes tactical gameplay, requiring players to manage limited resources, and coordinate movement carefully to stay alive. Each match gives the player a new opportunity to try new strategies, obtain new weapons, and break their high score!

Controls

  • WASD – Move
  • Mouse – Aim / Shoot
  • F – Interact
  • Shift – Run
  • Escape – Pause
